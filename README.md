# Passive Security Research

This repository contains responsibly documented, passive cybersecurity research.
All work here is performed ethically and legally using publicly available information.

## Scope
- Passive observation only
- No scanning, exploitation, or intrusive testing
- Documentation for learning and responsible disclosure

## Purpose
This repository exists to document learning-based security research,
public surface analysis, and responsible security observations.

## Ethics
All research documented here is non-intrusive and respects target systems.
No private data is accessed, modified, or exploited.
